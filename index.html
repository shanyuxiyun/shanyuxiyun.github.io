常用命令集合

0x01 新建一个网站

如果没有设置 folder ，Hexo 默认在目前的文件夹建立网站

0x02 新建一篇文章

新建一篇文章。如果没有设置 layout 的话，默认使用 _config.yml 中的 default_layout 参数代替。
如果标题包含空格的话，请使用引号括起来

0x03 生成静态文件

-d, –deploy 文件生成后立即部署网站
-w, –watch 监视文件变动

0x04 发表草稿

0x05 启动本地服务

-p, –port 重设端口
-s, –static 只使用静态文件
-l, –log 启动日记记录，使用覆盖记录格式

0x06 部署服务

hexo deploy

-g, –generate 部署之前预先生成静态文件

使用中遇到的问题

1. Error: bash: /dev/tty: No such device or address

配置_config.xml属性时候，冒号后面必须要有一个空格

2. git部署没有错误提示，但是github访问却没有更新内容

配置**_config.xml**deploy仓库的时候，仓库名必须为【your_user_name.github.io】，固定写法

3. ERROR Process failed: languages/default.yml archive_b: Archives: %s 

这里需要把所有模板中的archive_b:
Archives: %s => archive_b: "Archives: %s"
加啥双引号，不然会解析错误。
